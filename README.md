📌 Project Overview

This project evaluates the impact of a product feature using A/B testing and statistical analysis.

🎯 Business Problem

Companies need to validate whether new features improve user engagement and conversion before full rollout.

🛠️ Tools & Technologies
Python (Pandas, SciPy)
Statistical Analysis
Hypothesis Testing
Data Visualization

📊 Key Analysis
Conversion rate comparison
Hypothesis testing (t-test)
Statistical significance evaluation
Segment-level analysis

🔍 Key Insights
The variant group shows improved conversion compared to control
Results are statistically significant
Certain user segments respond better to changes

💡 Business Recommendations
Roll out feature if statistically significant
Test further across different segments
Continue experimentation for optimization
